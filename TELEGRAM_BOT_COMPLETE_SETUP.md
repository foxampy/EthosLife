# 📋 ЧАСТЬ 1: БЫСТРАЯ НАСТРОЙКА (10 минут)

## 1.1 Создание Бота

### Шаг 1: Открыть @BotFather
```
1. Открыть Telegram
2. Найти @BotFather
3. Нажать "Start"
```

### Шаг 2: Создать нового бота
```
Отправить команду: /newbot

BotFather попросит:
1. Name: EthosLife Health Bot
2. Username: ethoslife_health_bot (должен заканчиваться на _bot)
```

### Шаг 3: Скопировать токен
```
BotFather отправит:
Here is your token: 123456789:ABCdefGHIjklMNOpqrsTUVwxyz

Скопировать этот токен!
```

### Шаг 4: Настроить бота
```
Отправить @BotFather:
/setdescription
Выбрать бота
Описание: 🌱 EthosLife - Ваш персональный помощник здоровья

/setuserpic
Выбрать бота
Загрузить логотип EthosLife

/setcommands
Выбрать бота
Отправить команды:
start - Главное меню
help - Справка
reminders - Мои напоминания
schedule - Расписание
ai - AI-коуч
specialist - Специалист
center - Центр здоровья
progress - Прогресс
settings - Настройки
```

# 🔧 ЧАСТЬ 3: ТЕХНИЧЕСКАЯ НАСТРОЙКА

## 3.1 База Данных

### Таблица telegram_users
```sql
CREATE TABLE telegram_users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  chat_id BIGINT UNIQUE NOT NULL,
  first_name VARCHAR(255),
  last_name VARCHAR(255),
  username VARCHAR(255),
  language VARCHAR(10) DEFAULT 'ru',
  notifications_enabled BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  last_seen TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_telegram_users_chat_id ON telegram_users(chat_id);
```

### Таблица reminders
```sql
CREATE TABLE reminders (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES telegram_users(id) ON DELETE CASCADE,
  title VARCHAR(255) NOT NULL,
  description TEXT,
  time TIME NOT NULL,
  frequency VARCHAR(20) DEFAULT 'daily',
  active BOOLEAN DEFAULT true,
  reminder_type VARCHAR(50),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_reminders_user_id ON reminders(user_id);
CREATE INDEX idx_reminders_time ON reminders(time);
```

### Таблица messages
```sql
CREATE TABLE messages (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES telegram_users(id) ON DELETE CASCADE,
  specialist_id UUID,
  content TEXT NOT NULL,
  direction VARCHAR(10),
  read BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_messages_user_id ON messages(user_id);
```
