# 📊 PART 2: B2B ONBOARDING FUNNEL

## Overview

**Two Tracks:**
1. **Specialists** — Individual practitioners (nutritionists, psychologists, trainers, doctors)
2. **Centers** — Wellness clinics, fitness studios, yoga centers

**Target Metrics:**
- Registration → Profile Completion: 70%
- Profile Completion → First Booking: 50%
- Time to First Booking: < 7 days
- Monthly Retention: 85%
- Free → Paid Conversion: 15-20%
