## 🎯 КОНЦЕПЦИЯ ВОРОНКИ

### Принцип "Погружение до оплаты"

```
Внимание → Интерес → Желание → Доверие → Действие → Удержание
   ↓          ↓          ↓          ↓           ↓           ↓
Landing   Features   Preview   Social     Register   Subscribe
Page       (модули)   (данные)  Proof     (быстрая)  (бонусы)
```

**Ключевая идея:** Пользователь должен ПОЧУВСТВОВАТЬ ценность ДО регистрации.

## 📱 ЭТАП 2: ИНТЕРАКТИВНЫЙ PREVIEW (Интерес)

### Концепция: "Попробуй до регистрации"

**Страница:** `/preview/health`

```
┌──────────────────────────────────────────────────────────────┐
│  🔒 Демо-режим                                              │
│  Введите свои данные чтобы увидеть персональные рекомендации │
│  [Зарегистрироваться] [Продолжить в демо]                   │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  Ваш дневник питания сегодня                                 │
│  ───────────────────────────                                 │
│                                                              │
│  Завтрак: [_______________] [+ Добавить]                    │
│  Калории: 0 / 2000 kcal                                     │
│                                                              │
│  💡 Рекомендация AI:                                        │
│  "Добавьте белка на завтрак для лучшей концентрации"        │
│                                                              │
│  [💾 Сохранить] ← Требует регистрации                       │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

**Логика:**
1. Пользователь вводит данные (например, что съел на завтрак)
2. AI даёт рекомендацию
3. Пользователь хочет сохранить → видит "Зарегистрируйтесь"
4. Мотивация: "Уже ввели данные — не хотите потерять?"

## 💡 КЛЮЧЕВЫЕ ПРИНЦИПЫ

### 1. Меньше трения
- Регистрация в 1 клик (Google/Telegram)
- Не запрашивать лишние данные
- Сохранять прогресс

### 2. Больше ценности до оплаты
- Персональный план виден ДО регистрации
- AI рекомендации работают в демо
- Можно "пощупать" все модули

### 3. Социальное доказательство
- Отзывы на каждом этапе
- Счётчики ("47 бонусов осталось")
- Истории успеха

### 4. Срочность и дефицит
- Таймер на бонусе
- Ограниченное количество
- "Только сегодня"
