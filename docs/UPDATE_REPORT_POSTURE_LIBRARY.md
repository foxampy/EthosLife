## ✅ 2. Трекер осанки (Posture Tracker) - НОВОЕ

### Файлы:
- `client/src/components/PostureTracker.tsx` - основной компонент
- `client/src/pages/Posture.tsx` - страница

### Технологии:
- **MediaPipe Pose** - отслеживание ключевых точек тела
- **Camera Utils** - доступ к камере
- **Canvas API** - визуализация скелета

### Функционал:

#### 📹 Отслеживание камерой:
- Доступ к веб-камере (с разрешения)
- Распознавание ключевых точек тела в реальном времени (30 FPS)
- Визуализация скелета
- Зеркальное отображение для удобства

#### 📊 Анализ осанки:
- **Угол шеи** - отклонение от вертикали
- **Уровень плеч** - разница между левым и правым
- **Угол спины** - наклон позвоночника
- Определение "хорошая/плохая" осанка

#### 🔔 Система алертов:
- Звуковой сигнал (настраиваемый)
- Визуальные уведомления
- Настраиваемый интервал (10-300 сек)
- Счетчик нарушений
- Пороговые значения для срабатывания

#### 💼 Рекомендации по рабочему месту:
- Положение монитора (уровень глаз, расстояние)
- Настройка стула (высота, угол коленей)
- Правильная посадка (спина, плечи, шея)

#### 🏃 Разминки:
Три типа упражнений:
1. **Сидя** (4 упражнения)
   - Растяжка шеи
   - Вращение плечами
   - Скручивание позвоночника
   - Растяжка запястий

2. **Стоя** (3 упражнения)
   - Стеновые ангелы
   - Растяжка подвздошных мышц
   - Раскрытие грудной клетки

3. **С ковриком** (4 упражнения)
   - Кошка-Корова
   - Поза ребенка
   - Поза кобры
   - Нить в игольном ушке

Каждое упражнение включает:
- Название и описание
- Длительность
- Ссылка на YouTube видео
- Фото (placeholder)
- Для каких состояний подходит

### Приватность:
- Все данные обрабатываются локально
- Видео НЕ отправляется на сервер
- НЕ сохраняется на диск
- Требуется разрешение на камеру

## 🚀 Как использовать

### Библиотека:
1. Открыть `/library`
2. Просмотреть проверенные базы данных
3. Нажать "+" чтобы добавить в свою библиотеку
4. Нажать "Открыть" для перехода на сайт базы

### Трекер осанки:
1. Открыть `/posture`
2. Нажать "Старт" и разрешить доступ к камере
3. Сесть так, чтобы было видно верх тела
4. Следить за индикаторами осанки
5. Получать алерты при нарушении
6. Выбирать разминки из списка

### Интеграции:
1. Открыть `/integrations`
2. Выбрать категорию (носимые устройства, питание, etc.)
3. Нажать "Подключить" для интересующей интеграции
4. Настроить синхронизацию
