**Документ:** `SALES_FUNNEL_DESIGN.md` (32.7 KB)

```
Landing Page → Preview → Онбординг → Персональный план → Регистрация → Upsell
```

1. **Hero Section** с оффером и таймером
2. **Interactive Preview** — попробуй до регистрации
3. **5-шаговый онбординг** с генерацией плана
4. **Оффер:** Бесплатный месяц + консультация 6 специалистов
5. **Email воронка** (6 писем)
6. **Upsell** после регистрации (годовая подписка -50%)

#### Целевая конверсия:
- Регистрация: **15%** (vs текущих 3%)
- Подписка: **20%** от регистраций
- Итог: **3%** платящих пользователей
