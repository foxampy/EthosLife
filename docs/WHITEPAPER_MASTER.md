## 1. ВВЕДЕНИЕ

### 1.1 Что такое EthoLife?

**EthoLife** — комплексная цифровая экосистема управления здоровьем, объединяющая физическое, психологическое и социальное благополучие человека в единой платформе с AI-помощником.

> *Название происходит от греческого "ethos" (образ жизни, характер) + "life" (жизнь)*

### 1.2 Миссия

Сделать управление здоровьем доступным, персонализированным и мотивирующим для каждого человека, создав единую экосистему, где все аспекты здоровья работают вместе.

### 1.3 Видение

Создать глобальную платформу здоровья №1 в мире, объединяющую пользователей, специалистов и исследователей для достижения общей цели — здорового и полноценного общества.

## 2. ПРОБЛЕМА И РЕШЕНИЕ

### 2.1 Проблемы рынка

| Проблема | Описание |
|----------|----------|
| **Фрагментация** | Пользователи используют 5-10 разных приложений для разных аспектов здоровья |
| **Отсутствие связности** | Данные из разных приложений не интегрируются |
| **Высокая стоимость** | Качественные AI-ассистенты (ChatGPT Plus) стоят $20/мес |
| **Нет мотивации** | Статичные приложения не мотивируют на постоянство |
| **Сложность доступа к специалистам** | Нет единой платформы для поиска и работы со специалистами |

### 2.2 Решение EthoLife

| Решение | Как работает |
|---------|--------------|
| **7 модулей в одном** | Все аспекты здоровья в единой платформе |
| **AI-помощник бесплатно** | Qwen/Groq/Gemini вместо платного ChatGPT |
| **Unity Token** | Геймификация с реальными вознаграждениями |
| **Маркетплейс специалистов** | Прямой доступ к врачам, тренерам, психологам |
| **Социальная поддержка** | Сообщество единомышленников и групповые челленджи |

### 3.4 Unity Token (UNITY)

**Тип:** Utility token (офчейн)  
**Курс:** 1 USD = 8.5 UNITY  
**Назначение:** Геймификация и оплата услуг

**Как получить:**
| Действие | Награда |
|----------|---------|
| Регистрация | 100 UNITY |
| Приглашение друга | 500 UNITY |
| Стрик 7 дней | 50 UNITY |
| Стрик 30 дней | 300 UNITY |
| Кэшбэк Basic | 50/мес |
| Кэшбэк Premium | 150/мес |
| Покупка подписки рефералом | 1000 UNITY |

**Где использовать:**
- Оплата подписок (скидка 15%)
- Доступ к премиум-функциям
- Вознаграждение рефералов
- Голосование в DAO (в будущем)

```
Новые токены = Активность пользователей × Коэффициент вознаграждения
```
